# Cyber Security Toolkit
### By Jayesh Goyal | SDJ International College

A collection of cybersecurity tools.

## Tools Included:

### 1. Password Checker
- Analyzes password strength
- Gives improvement tips
- Scores password 1-10

### 2. Port Scanner
- Scans common ports on any IP
- Identifies running services
- Detects dangerous open ports

### 3. Website Info Grabber
- WHOIS lookup
- DNS records
- IP resolution
- Open port detection

### 4. Email Header Analyzer
- Detects email spoofing
- Traces mail servers
- Identifies phishing indicators

### 5. Intrusion Detection System
- Auto detects WiFi/Hotspot network
- Monitors for new devices
- Alerts on suspicious ports
