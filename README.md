## 🛠️ Tools In This Repository

### 1. 🔐 Password Checker
Analyzes password strength and gives improvement tips.
Scores password on 5 parameters.

### 2. 🌐 Port Scanner
Scans common ports on any IP address.
Identifies running services and dangerous open ports.

### 3. 🔍 Website Info Grabber
- WHOIS lookup
- DNS records
- IP resolution
- Open port detection

### 4. 📧 Email Header Analyzer
- Detects email spoofing
- Traces mail servers
- Identifies phishing indicators

### 5. 🛡️ Intrusion Detection System
- Auto detects WiFi/Hotspot network
- Monitors for new devices joining
- Alerts on suspicious open ports
- Works on WiFi + Hotspot + Mobile data

### 6. 🔐 Crypto Toolkit
- MD5, SHA1, SHA256, SHA512 hashing
- Base64 encode/decode
- Caesar cipher encryption
- File integrity verification

### 7. 🌐 Subdomain Finder
- Finds subdomains of any domain
- Multi-threaded for speed
- Used in OSINT investigations

### 8. 📸 Image Metadata Extractor
- Extracts EXIF data from images
- Finds hidden GPS location data
- Identifies device used to take photo

### 9. 🔍 File Integrity Monitor
- Creates baseline hash of files
- Detects modified/deleted/new files
- Real-time monitoring mode
- Used in digital forensics

### 10. 🤖 Auto Recon Tool
- Full domain reconnaissance
- IP, WHOIS, DNS, Ports, Headers, SSL
- One command does everything
